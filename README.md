# 💻 devTinder Frontend

This is the client-side application for devTinder, built with React and Redux Toolkit. It provides a smooth, responsive interface for developers to discover and chat with each other.

## 🛠️ Tech Stack
* **Framework:** React 18+
* **State Management:** Redux Toolkit (for user and feed management)
* **Styling:** Tailwind CSS & DaisyUI
* **Routing:** React Router DOM
* **Real-time:** Socket.io-client
* **HTTP Client:** Axios

## 🚀 Features
* **Live Profile Preview:** Instant feedback while editing profile details.
* **Real-time Chat:** Instant messaging with connection-based access.
* **Interactive Feed:** Swipe-like functionality to connect or pass on profiles.
* **Responsive Design:** Optimized for both desktop and mobile views.
